# **App Name**: Solo Leveling System

## Core Features:

- User Onboarding: Collect user data (Name, Age, Weight, Height, Training Frequency) during initial setup.
- Hunter Dashboard: Display user Rank, Level, XP bar, and Shadow Avatar.
- Stat Chart: Visualize STR, VIT, AGI, INT, and PER stats using a radar chart.
- Daily Quest Generation: Generate daily exercise quests (Flexões, Abdominais, Agachamentos, Corrida) dynamically adjusted based on user data. As a tool, calculate scaling appropriately using all relevant biometric parameters.
- Quest Log: Present exercise quests as cards with neon checkboxes and progress bars.
- Progression System: Calculate XP based on quest difficulty and consistency. Determine Rank (E to S) and Level based on XP.
- Penalty Quest: Upon quest failure, trigger a 'Penalty Quest' (extra Burpees or light activity) to enforce consistency.

## Style Guidelines:

- Primary color: Neon Blue (#00D4FF) to represent the System's interface and power.
- Background color: Absolute Dark Mode (#050505) for a high-contrast, immersive experience.
- Accent color: Dark Purple (#1A0033) to complement the blue and maintain a dark aesthetic.
- Body and headline font: 'Space Grotesk', a sans-serif for a modern and tech-oriented feel.
- Neon-style icons with a futuristic and minimalist design.
- Mobile-first layout with a focus on readability and ease of navigation.
- Subtle glow effects and animations to enhance the neon aesthetic.
