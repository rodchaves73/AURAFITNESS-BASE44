
# Aura System AI - Solo Leveling Edition

Este é o seu web app **Aura System**, uma plataforma imersiva de fitness inspirada no universo de Solo Leveling.

## 📥 Como Baixar o Código (IMPORTANTE)

1. No topo da tela, clique no nome do projeto **"Aura Fitness"**.
2. Selecione **"Export Project"** ou **"Download ZIP"**.
3. Extraia o conteúdo e abra no seu editor (ex: VS Code).

## 📱 Como colocar o Ícone Oficial (Selo AF)

Para que o PWABuilder reconheça seu app e ele apareça no seu celular:
1. **Salve a imagem** do ícone (Selo AF).
2. No menu de pastas à esquerda, localize a pasta **`public`** (está na raiz do projeto).
3. Faça o upload do arquivo para dentro desta pasta.
4. **MUITO IMPORTANTE:** O nome do arquivo deve ser exatamente `icon.png` (letras minúsculas). 

## 🎵 Como Adicionar Meus Próprios Sons

Para usar seus próprios efeitos sonoros:
1. No menu de pastas, vá para a pasta **`public`**.
2. Crie uma pasta chamada **`sounds`** (opcional, mas recomendado).
3. Faça o upload dos seus arquivos `.mp3` ou `.ogg`.
4. **Informe o nome do arquivo ao Arquiteto** para que ele atualize o código de áudio para você. Exemplo: `/sounds/meu-efeito.mp3`.

## 🚀 Como rodar localmente

1. `npm install`
2. Configure sua `GOOGLE_GENAI_API_KEY` no `.env`.
3. `npm run dev`

**Arise.**
